# Random SSN

A random Social Security Number generator

## Dependencies

* node_modules - `yarn` or `npm install`
* Foreman - `yarn global add foreman` or `npm install -g foreman`

## Running the tests

```bash
yarn test
```

## Running

* Start the project

```bash
nf run yarn start
```

* Make a request

```bash
curl http://localhost:3000/
```
